Bu çalışma, OpenWeatherMap API kullanılarak geliştirilmiş, gerçek zamanlı hava durumu verilerini sunan dinamik bir web uygulamasıdır. Kullanıcının girdiği şehir ismine göre hava durumunu analiz eder ve görsel bir deneyim sunar.

  - Öne Çıkan Özellikler

Canlı Veri Entegrasyonu: OpenWeatherMap API aracılığıyla dünya genelindeki şehirlerin güncel sıcaklık, nem ve hava durumu verilerini çeker. 



Dinamik Tema Yönetimi: API'den gelen hava durumu koduna göre arka plan rengini ve hava durumu ikonlarını otomatik olarak günceller. 


Hızlı Arama: Kullanıcı dostu arayüzü sayesinde şehir bazlı hızlı sorgulama ve anlık sonuç görüntüleme imkanı sağlar. 

  - Teknik Detaylar

Asenkron Programlama: Veri çekme işlemleri modern JavaScript yöntemlerinden olan Fetch API ve Async/Await yapısı kullanılarak optimize edilmiştir. 




Veri İşleme (JSON): API'den gelen karmaşık JSON verileri parse edilerek kullanıcı arayüzüne (DOM) dinamik olarak yansıtılır. 


Hata Yönetimi: Geçersiz şehir girişleri veya ağ bağlantı sorunları gibi durumlarda kullanıcıya uygun geri bildirimler verilir.

  - Kullanılan Teknolojiler

JavaScript (ES6+): API istekleri ve mantıksal kurgu. 


HTML5 & CSS3: Responsive tasarım ve modern görsel arayüz. 


OpenWeather API: Gerçek zamanlı hava durumu veri kaynağı. 

Kurulum ve Kullanım
Proje dosyalarını bilgisayarınıza indirin.

Tarayıcınızda main.html (veya index.html) dosyasını açın.

Merak ettiğiniz bir şehrin ismini yazın ve hava durumunu anında görün!
